mermery.js

Improve form state recovery with this minimal jQuery plugin that stores field values in HTML5 sessionStorage in any moment and/or situation.

A Quick Example

The example below will store form values when page loaded so that you could enter some data and recover the initial state anytime pushing the "Unstash Form" button. The plugin clears session data when the user push the "Clear data" button or submits the form, otherwise it will last until the browser window is closed.
